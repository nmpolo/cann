nmpolo-cann
===========

Requirements
------------

- [nodejs](https://github.com/joyent/node/wiki/Installation)
- [bower](https://github.com/bower/bower)
- [grunt](http://gruntjs.com/getting-started)

Installation
------------

### Install node modules and bower assets
```bash
npm install
bower install
```

### Build everything into dist/
```bash
grunt
```
